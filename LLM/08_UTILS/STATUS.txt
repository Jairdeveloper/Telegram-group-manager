💡 CARACTERÍSTICAS NUEVAS
==========================

1. Pattern Engine Optimizado
   • Regex compilation + caché
   • Variable binding
   • Case-insensitive matching
   • Timeout protection

2. Búsqueda Semántica
   • semantic_search(query, candidates, top_k=5)
   • Similitud coseno entre textos
   • Clustering automático
   • Basado en Sentence Transformers

3. LLM Fallback Automático
   • Intenta OpenAI (si key disponible)
   • Fallback a Ollama local
   • Sin configuración manual
   • Temperature + max_tokens ajustables

4. Persistencia Multi-Tabla
   • Conversations (historial)
   • Knowledge (patrones)
   • Embeddings (vectores)
   • Users (perfiles)

5. REST API Completa
   POST   /api/v1/chat              → Chat
   GET    /api/v1/chat/history/{id} → Historial
   GET    /api/v1/stats/user/{name} → Stats
   GET    /api/v1/health            → Health check

6. Configuration Management
   • Centralizado en src/core/config.py
   • Pydantic validation
   • Environment variables
   • Defaults sensatos
