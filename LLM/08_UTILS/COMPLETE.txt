🚀 GETTING STARTED (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

Step 1: Install
  cd chatbot_evolution
  pip install -e .

Step 2: Run
  # CLI Mode
  python -m src.main --mode cli

  # OR API Mode
  python -m src.main --mode api
  # → http://localhost:8000/docs

  # OR Docker
  docker-compose up

Step 3: Test
  pytest tests/ -v
  # → 50+ tests pass
