🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALACIÓN (1 minuto)
    cd chatbot_evolution
    pip install -e .

2️⃣  CLI INTERACTIVO (Prueba inmediata)
    python -m src.main --mode cli
    > hello
    Bot: Hello there how can I help?
    > (quit)

3️⃣  REST API (Desarrollo)
    python -m src.main --mode api
    → http://localhost:8000/docs (Swagger UI)

4️⃣  TESTS (Validación)
    pytest tests/ -v
    → 50+ tests ejecutados

5️⃣  DOCKER (Producción)
    docker-compose up --build
    → http://localhost:8000
